# شفيع — مساعد بحث قرآني

مساعد ذكاء اصطناعي موثوق للبحث القرآني والتفسير والإرشاد.

## التشغيل

```bash
npm install
npm run dev
```

## النشر على Vercel

1. ارفع على GitHub
2. vercel.com/new → استورد الريبو
3. أضف متغير البيئة: `GEMINI_API_KEY`
4. Deploy

المفتاح من: https://aistudio.google.com/apikey
